##  Key Features

- Stress level prediction using a **Random Forest Classifier**
- Clean and interactive **Streamlit UI**
- Color-coded stress level output
- Personalized **stress management tips**
- Offline model evaluation using standard ML metrics

---

##  Machine Learning Overview

- **Problem Type:** Multi-class Classification  
- **Target Variable:** Stress Level (Low, Medium, High)  
- **Algorithm Used:** Random Forest Classifier  
- **Why Random Forest?**
  - Better generalization than a single decision tree
  - Handles non-linear patterns effectively
  - Suitable for real-world ML applications

##  How to Run the Project

1. Clone the repository from GitHub to your local system.
2. Open the project folder in a code editor like VS Code.
3. Install the required Python dependencies using the requirements file.
4. Run the Streamlit application file (`app.py`).
5. Open the local URL shown in the terminal to access the application in your browser.
